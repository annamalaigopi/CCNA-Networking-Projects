## 📂 Projects

### 01. Multi-Branch Enterprise Network

**Technologies Used:**
- DHCP
- DNS
- Web Server
- Routing
- LAN and WAN Connectivity

**Project File:**  
`01-Multi-Branch-Network-DHCP-DNS-Web-Server.pkt`

**Description:**

Designed and configured a multi-branch enterprise network with DHCP for automatic IP address allocation, DNS for name resolution, and Web Server services. Tested end-to-end connectivity between different networks.

---

### 02. Enterprise VLAN Network

**Technologies Used:**
- VLANs
- Trunking
- Inter-VLAN Routing
- Multilayer Switch
- DHCP
- DNS
- Web Servers
- EtherChannel
- PAgP
- LACP

**Project File:**  
`02-Enterprise-VLAN-InterVLAN-EtherChannel.pkt`

**Description:**

Designed an enterprise VLAN network using a multilayer switch for Inter-VLAN Routing. Configured VLANs for different departments and implemented EtherChannel using PAgP and LACP for link aggregation and redundancy.

### 03. Enterprise WAN Network with NAT and PAT

**Technologies Used:**
- Dynamic NAT
- Static NAT
- PAT (NAT Overload)
- DNS
- Web Servers
- WAN Connectivity

**Project File:**  
`03-Enterprise-WAN-NAT-PAT-DNS-Web-Servers.pkt`

**Description:**

Implemented an enterprise WAN network with Dynamic NAT, Static NAT, and PAT to provide private-to-public IP address translation and access to network services across multiple networks.

### 04. Enterprise WAN with VPN and GRE Tunnel

**Technologies Used:**
- VPN Concepts
- GRE Tunnel
- RIP
- NAT
- DNS
- Web Servers
- WAN Routing

**Project File:**  
`04-Enterprise-WAN-VPN-GRE-Tunnel-NAT-RIP.pkt`

**Description:**

Configured a GRE tunnel between remote enterprise networks to provide logical site-to-site connectivity across a WAN. Implemented dynamic routing and network services to provide end-to-end communication between remote sites.
